Variáveis de ambiente configuram apps sem precisar reconstruí-los.

## Objetivo

Manter segredos seguros e deploys portáteis.

## Quando usar

Use variáveis de ambiente para URLs de banco, chaves de API, feature flags e credenciais de serviços.

## Exemplo

```json
{
  "envVars": {
    "NODE_ENV": "production",
    "DATABASE_URL": "postgresql://user:pass@host:5432/db"
  }
}
```

## Regras

- Não commite segredos.
- Use nomes de variáveis claros.
- Rotacione chaves vazadas imediatamente.
- Mantenha valores de produção e desenvolvimento separados.
