Um workspace é o limite do seu time e recursos.

## Objetivo

Entender como os recursos são organizados.

## Quando usar

Use workspaces para separar projetos pessoais, times, clientes ou ambientes.

## Exemplo

```bash
curl https://api.hyze.cloud/api/apps/ \
  -H "Authorization: Bearer hyze_sua_chave"
```

## O que pertence a um workspace

- Apps
- Bancos de dados
- Membros do time
- Convites
- Planos e uso
- Chaves de API
