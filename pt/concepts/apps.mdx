Um app é a unidade principal que você deploya na Hyze Cloud.

## Objetivo

Entender o que um app contém antes de fazer deploy.

## Quando usar

Use apps para APIs, workers, dashboards, bots e serviços em background.

## Exemplo

```json
{
  "name": "api",
  "runtime": "node",
  "memoryMB": 512,
  "startupCommand": "npm start",
  "exposePort": 3000
}
```

## O que um app inclui

- Configuração de runtime
- Variáveis de ambiente
- Logs
- Acesso ao sistema de arquivos
- Backups
- Histórico de builds
