SDKs devem tornar fluxos comuns da Hyze Cloud nativos no seu código.

## Objetivo

Escolher o caminho de integração certo.

## Quando usar

Use um SDK quando quiser helpers de requisição tipados, erros consistentes e menos boilerplate.

## Status atual

A REST API é a fonte da verdade hoje.

As páginas de SDK documentam padrões de client recomendados até que pacotes oficiais sejam publicados.

<Note>
  Mantenha integrações de produção na REST API se você precisa de cobertura completa de endpoints agora.
</Note>
