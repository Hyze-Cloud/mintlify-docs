SDKs deixam os fluxos da Hyze Cloud nativos no seu código — helpers tipados, erros consistentes e menos boilerplate.

## Objetivo

Escolher o caminho certo de integração.

## Quando usar

Use um SDK quando quiser helpers tipados, erros padronizados e menos código repetido.

## Pacotes oficiais

| Pacote | Runtime | Status |
| --- | --- | --- |
| [`@hyzecloud/sdk`](/pt/sdks/typescript) | Node.js 18+, Bun | Disponível |

A REST API continua sendo a fonte da verdade. A SDK cobre as superfícies principais (apps, databases, keys, invoices, GitHub, plans).
