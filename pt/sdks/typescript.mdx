## Objetivo

Criar um helper de API reutilizável.

## Exemplo

```ts
const HYZE_API = "https://api.hyze.cloud/api";

async function hyze<T>(path: string, init?: RequestInit): Promise<T> {
  const res = await fetch(`${HYZE_API}${path}`, {
    ...init,
    headers: {
      Authorization: `Bearer ${process.env.HYZE_API_KEY}`,
      "Content-Type": "application/json",
      ...init?.headers,
    },
  });

  if (!res.ok) {
    throw new Error(`Hyze API falhou: ${res.status}`);
  }

  return res.json() as Promise<T>;
}

const currentPlan = await hyze("/plans/current");
```
