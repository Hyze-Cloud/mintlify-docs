Do zero até uma requisição autenticada funcionando.

## 1. Criar uma API key

1. Abra [app.hyze.cloud](https://app.hyze.cloud)
2. Vá em **Settings → API Keys → Create Key**
3. Copie a chave (`hyze_...`) — ela só aparece uma vez

## 2. Chamar a API

Base URL: `https://api.hyzecloud.com/api`

```bash
export HYZE_API_KEY="hyze_sua_chave"

curl https://api.hyzecloud.com/api/apps/ \
  -H "Authorization: Bearer $HYZE_API_KEY"
```

Uma lista JSON de apps significa que a autenticação está ok.

### Com a SDK

```bash
npm install @hyzecloud/sdk
```

```ts
import { HyzeCloud } from "@hyzecloud/sdk";

const hyze = new HyzeCloud({ apiKey: process.env.HYZE_API_KEY });
const { apps } = await hyze.apps.list();
console.log(apps);
```
