A API é a principal superfície de automação hoje.

## Objetivo

Entender onde fluxos de CLI se encaixam.

## Quando usar

Use scripts estilo CLI para CI/CD, deploys locais, smoke tests e tarefas operacionais.

## Exemplo

```bash
#!/usr/bin/env bash
set -euo pipefail

curl https://api.hyzecloud.com/api/apps/ \
  -H "Authorization: Bearer $HYZE_API_KEY"
```

## Configuração recomendada

- Armazene `HYZE_API_KEY` no seu shell ou gerenciador de segredos do CI.
- Mantenha scripts idempotentes.
- Exiba mensagens curtas de sucesso e falha.
