Logs ajudam a depurar falhas de inicialização, crashes e comportamento em runtime.

## Objetivo

Obter linhas de log recentes de um app.

## Quando usar

Use logs após deploys, reinicializações, mudanças de ambiente ou relatos de incidentes.

## Exemplo

```bash
curl "https://api.hyze.cloud/api/apps/app_001/logs?tail=200&timestamps=true" \
  -H "Authorization: Bearer hyze_sua_chave"
```

## Opções

| Query | Padrão | Descrição |
| --- | --- | --- |
| `tail` | `100` | Número de linhas a retornar. |
| `timestamps` | `false` | Incluir timestamps na saída. |
