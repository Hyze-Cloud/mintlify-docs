## Deploy a partir de um ZIP

<Steps>
  <Step title="Empacote seu app">
    Crie um ZIP com os arquivos que a Hyze Cloud deve executar. Inclua o entrypoint do seu app e o manifesto de depend횄짧ncias.
  </Step>
  <Step title="Envie a requisi횄짠횄짙o de deploy">
    ```bash
    curl -X POST https://api.hyzecloud.com/api/apps/deploy \
      -H "Authorization: Bearer hyze_sua_chave" \
      -F "file=@app.zip" \
      -F "name=my-api" \
      -F "runtime=node" \
      -F "memoryMB=512" \
      -F "startupCommand=node server.js" \
      -F "exposePort=3000"
    ```
  </Step>
  <Step title="Veja os logs">
    ```bash
    curl "https://api.hyzecloud.com/api/apps/app_001/logs?tail=200&timestamps=true" \
      -H "Authorization: Bearer hyze_sua_chave"
    ```
  </Step>
</Steps>

  "autoDeploy": true,
  "repository": {
    "id": "repo_001",
    "owner": "hyzecloud",
    "name": "my-api",
    "branch": "main"
