title: Deploy pelo GitHub
description: Faça deploy de um app a partir de um repositório GitHub conectado.
---

Use deploys por GitHub quando seu código-fonte já estiver em um repositório.

## Objetivo

Criar um app a partir de um repositório e opcionalmente ativar auto deploys.

## Quando usar

Use este fluxo para apps de produção, workflows em equipe e deploys por branch.

## Exemplo

```bash
curl -X POST https://api.hyze.cloud/api/apps/deploy-from-repo \
  -H "Authorization: Bearer hyze_sua_chave" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "api",
    "runtime": "node",
    "memoryMB": 512,
    "startupCommand": "npm start",
    "exposePort": 3000,
    "autoDeploy": true,
    "repository": {
      "owner": "hyzecloud",
      "name": "api",
      "branch": "main"
    }
  }'
```

## Como funciona

A Hyze Cloud baixa a branch selecionada, faz o build do app e inicia com seu `startupCommand`.

<Tip>
  Ative `autoDeploy` para que pushes na branch disparem novos deploys automaticamente.
</Tip>
