SDKs make common Hyze Cloud workflows feel native in your codebase — typed helpers, consistent errors, less boilerplate.

## Goal

Choose the right integration path.

## When to use

Use an SDK when you want typed request helpers, consistent errors, and less boilerplate.

## Official packages

| Package | Runtime | Status |
| --- | --- | --- |
| [`@hyzecloud/sdk`](/en/sdks/typescript) | Node.js 18+, Bun | Available |

The REST API remains the source of truth. The SDK covers the main product surfaces (apps, databases, keys, invoices, GitHub, plans).
