Use the official **`@hyzecloud/sdk`** package on Node.js 18+ or Bun.

## Install

```bash
npm install @hyzecloud/sdk
# or
bun add @hyzecloud/sdk
```

## Quickstart

```ts
import { HyzeCloud, HyzeError } from "@hyzecloud/sdk";

const hyze = new HyzeCloud({
  apiKey: process.env.HYZE_API_KEY, // hyze_...
});

const { apps } = await hyze.apps.list();
console.log(apps);

try {
  await hyze.apps.restart("app_001");
} catch (err) {
  if (err instanceof HyzeError) {
    console.error(err.status, err.code, err.message);
    if (err.isRateLimited) {
      console.error("Retry after", err.retryAfterSeconds, "s");
    }
  }
}
```

## Configuration

| Option | Default | Description |

## Common workflows

### Deploy a ZIP

```ts
import { readFileSync } from "node:fs";
import { HyzeCloud } from "@hyzecloud/sdk";

const hyze = new HyzeCloud({ apiKey: process.env.HYZE_API_KEY });

await hyze.apps.deployFromZip({
  file: readFileSync("./app.zip"),
  name: "my-api",
  runtime: "node",
  memoryMB: 512,
  startupCommand: "node server.js",
  exposePort: 3000,
});
```

### Provision a database

```ts
await hyze.databases.create({
  name: "prod-postgres",
  engine: "postgresql",
  memoryMB: 1024,
  storageGB: 20,
});
```

### Logs and env

```ts
await hyze.apps.logs("app_001", { tail: 200, timestamps: true });
await hyze.apps.setEnv("app_001", { NODE_ENV: "production" });
```

## Resources

| Resource | Examples |
| --- | --- |

## Low-level requests

```ts
await hyze.get("/apps/");
await hyze.post("/apps/app_001/restart");
await hyze.request("/apps/", {
  method: "GET",
  query: { workspaceId: "org_1" },
});
```

## Errors

`HyzeError` is thrown on non-2xx responses:

| Property | Description |
| --- | --- |
| `status` | HTTP status |
| `code` | API error code when present |
| `message` | Human-readable message |
| `body` | Parsed response body |
| `retryAfterSeconds` | From `Retry-After` (rate limits) |
| `isRateLimited` | `status === 429` |
| `isUnauthorized` | `401` / `403` |
| `isNotFound` | `404` |
