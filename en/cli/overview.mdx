The API is the primary automation surface today.

## Goal

Understand where CLI workflows fit.

## When to use

Use CLI-style scripts for CI/CD, local deploys, smoke tests, and operational tasks.

## Example

```bash
#!/usr/bin/env bash
set -euo pipefail

curl https://api.hyzecloud.com/api/apps/ \
  -H "Authorization: Bearer $HYZE_API_KEY"
```

## Recommended setup

- Store `HYZE_API_KEY` in your shell or CI secret manager.
- Keep scripts idempotent.
- Print short success and failure messages.
