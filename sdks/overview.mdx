SDKs should make common Hyze Cloud workflows feel native in your codebase.

## Goal

Choose the right integration path.

## When to use

Use an SDK when you want typed request helpers, consistent errors, and less boilerplate.

## Current status

The REST API is the source of truth today.

SDK pages document recommended client patterns until official packages are published.

<Note>
  Keep production integrations on the REST API if you need full endpoint coverage now.
</Note>
